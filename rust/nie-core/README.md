# nie-core (public stub)

This directory in the public repository is a stub. The real nie-core Cargo workspace
containing nie-crypto, nie-tpm, nie-token, nie-providence, nie-rbac, nie-enrollment,
nie-revocation, nie-wasm, nie-ffi, and nie-embedded lives in the private deployment
repository.

Integration contract between the Node broker and nie-core is documented at
`docs/nie-integration.md`. Only interface signatures and stated security properties
appear there. Construction details are not disclosed.
